# Instructions
Generate a personalized interpretation of the study findings for this specific patient. Create three distinct sections:

## 1. Study Summary (studySummary)
Write 1-2 sentences summarizing what the study found. Keep it simple and jargon-free.
Example: "A large study of 27,000 patients found that adding a new medication to standard statin therapy further reduced heart attack and stroke risk."

## 2. Patient-Specific Projection (patientProjection)
Apply the study's effect sizes to the patient's actual values when available. Use concrete numbers.
Example: "Your current LDL cholesterol of 150 mg/dL is similar to patients in this study. If you achieved the same reduction, your LDL could potentially drop to around 90 mg/dL."

## 3. Contextualized Risk (contextualizedRisk)
Frame the benefits using "N out of 100" language to make the numbers tangible.
Example: "In the study, among 100 patients like you treated for about 2 years, roughly 2 fewer experienced a heart attack or stroke compared to those on standard treatment alone."

# Tone Guidelines
- Conversational and empowering, not alarming
- Use "you" and "your" to personalize
- Include appropriate caveats ("In this study...", "Results may vary...")
- Avoid medical jargon or explain terms when necessary
- Be honest about limitations

# Output Format
Return your response as a JSON object:

{
  "studySummary": "1-2 sentence summary of the study",
  "patientProjection": "Personalized projection using patient's values",
  "contextualizedRisk": "Risk framed as 'N out of 100' with follow-up duration"
}

# Important Notes
- If specific patient values aren't available, use qualitative language
- Always mention the follow-up duration when discussing risk
- Don't make claims beyond what the study supports
- If the patient's values are better/worse than study population, note this
- Frame benefits positively but realistically

Return only the JSON object, no additional text.
